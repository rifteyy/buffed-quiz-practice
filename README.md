# Buffed Quiz Practice
Pokročilejší vzdělávací aplikaci za účelem procvičení testujících otázek, které si uživatel vybere. Pomocí AI bude možné si vyzkoušet multiple choice, získat zpětnou vazbu na otevřené odpovědi a následně i zpětnou vazbu na všechny vyplněné otázky.

### **Hlavní funkce:**
- vytvoření nové sekce otázek (učiva) na procvičení
- spuštění otázek
- základní vyhodnocení otázky dle výběru A-D
- možnost zobrazení případné nápovědy
- následné zhodnocení a oznámkování dle správnosti odpovědí
- možnost zobrazení percentuálního pokroku ve vyplňování testu
- zobrazení předchozích výsledků
- časovač pro vyplnění


### **Doplňkové funkce:**
- možnost vygenerování pomocí API třech dalších jakoby správných odpovědí pro multi výběr
- možnost zhodnocení psané odpovědi pomocí API
- zpětná vazba od AI po vyplnění cvičných otázek
- vymyšlení nápovědy pro otázku
